# Система управления объявлениями для модерации

Сделала веб-приложение для модерации объявлений, упрощённый вариант внутренней системы Авито.

## Что сделалa:

- Настроила проект на Vite + React 19, подключила TypeScript.  
- Сделала главную страницу (/list) с карточками объявлений: фото (placeholder), название, цена, категория, дата, статус, приоритет.  
- Добавила фильтры: статус, категория, диапазон цен + поиск по названию и сброс фильтров.  
- Реализовала сортировку (дата, цена, приоритет) и пагинацию (10 карточек на страницу, навигация, общее количество).  

### Сделала страницу детального просмотра (/item/:id):

- галерея (3+ изображения)  
- описание и характеристики  
- информация о продавце  
- история модерации (кто проверял, когда, решение, комментарий)  
- панель действий модератора: Одобрить / Отклонить / Вернуть на доработку.  
- навигация: назад к списку, следующее/предыдущее объявление.  

### Сделала страницу статистики (/stats):

- карточки с метриками (всего проверено, процент одобренных/отклонённых, среднее время проверки)  
- графики: активность по дням, распределение решений, проверка по категориям.  

### Добавила дополнительные фишки:

- горячие клавиши (A, D, ←, →, /)  
- bulk-операции (чекбоксы, массовое одобрение/отклонение)  
- сохранение фильтров, URL-синхронизация  
- тёмная тема  
- экспорт статистики в CSV 
- real-time обновления списка  
- плавные анимации (переходы, появление карточек, progress bar)  

### Технические штуки:

- UI: MUI, Framer Motion  
- State: Redux Toolkit  
- HTTP: Axios, React Query  
- Валидация кода: ESLint + Prettier   
- Дополнительно: Dayjs, FileSaver, PapaParse, Recharts  

## Что не делала:

- Авторизацию/регистрацию  
- Загрузку реальных изображений  
- Интеграцию с внешними сервисами
